# ⚙️ AI Development Workflow

## 🔁 Ciclo padrão

1. Definir tarefa (TASK_TEMPLATE)
2. Enviar para IA
3. Revisar resposta
4. Ajustar se necessário
5. Implementar
6. Testar manualmente
7. Commit

---

## 🧠 Regra de ouro

Nunca aceitar código da IA sem entender.

---

## 🔍 Validação obrigatória

Antes de aceitar:

- O código faz sentido?
- Está simples?
- Está desacoplado?
- Vai escalar?

---

## 📦 Commits

Sempre usar padrão:

feat: add player movement  
fix: correct camera position  
refactor: separate controls logic  

---

## 🚀 Evolução do projeto

Ordem obrigatória:

1. Cena básica
2. Player
3. Movimento
4. Câmera
5. Interação
6. Dados reais

---

## ⚠️ Evitar

- Pular etapas
- Copiar código sem entender
- Depender 100% da IA
