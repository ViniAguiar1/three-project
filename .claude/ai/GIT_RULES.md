# 🧾 Git Rules

## 📌 Branches

- feat/*
- fix/*
- refactor/*
- chore/*

---

## 📦 Commits

Formato:

tipo: descrição

Ex:
feat: add player movement  
fix: correct keyboard input bug  

---

## 🚫 Evitar

- commits genéricos ("update", "fix")
- código quebrado na main

---

## 🔥 Regra principal

Cada commit deve representar uma evolução clara.
