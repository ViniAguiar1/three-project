# 🧩 Feature System

## 🎯 Objetivo

Permitir evolução controlada do projeto com features isoladas.

---

## 📌 Estrutura

Cada feature deve:

- Ter responsabilidade única
- Ser ativável/desativável
- Não quebrar outras partes

---

## 📦 Exemplo de features

- player-movement
- camera-follow
- object-interaction
- map-points
- ui-overlay

---

## 🧠 Estratégia

- Implementar feature por vez
- Validar antes de avançar
- Documentar cada feature

---

## 🔥 Regra principal

Uma feature não pode depender de outra mal definida.
