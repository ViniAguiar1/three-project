# 🧪 Code Quality Guidelines

## 📌 Regras gerais

- Código deve ser legível
- Nome de variáveis claro
- Funções pequenas
- Sem duplicação

---

## 🧠 3D específico

- Não recriar objetos em loop
- Usar useRef corretamente
- Evitar re-render

---

## ⚡ Performance

- Minimizar uso de estado global
- Evitar cálculos pesados no render
- Usar memo quando necessário

---

## 🧩 Estrutura

✔ Separar lógica de render  
✔ Separar input de lógica  
✔ Componentizar corretamente  

---

## 🚫 Evitar

- Funções gigantes
- Código mágico
- Side effects não controlados

---

## 🔥 Regra principal

Se não está claro, está errado.
