# 🔁 AI Task Pipeline

## 🎯 Pipeline

1. Definir task
2. Enviar para IA
3. Revisar output
4. Ajustar prompt
5. Implementar
6. Testar
7. Commitar

---

## 🧠 Validação

Antes de aceitar código:

- Está simples?
- Está modular?
- Está claro?
- Está alinhado com arquitetura?

---

## 🔄 Loop de melhoria

Task → IA → Ajuste → Código melhor → Repeat

---

## ⚠️ Anti-pattern

❌ Aceitar código direto  
❌ Não revisar  
❌ Não entender  

---

## 🔥 Regra principal

Você é o engenheiro. A IA é só ferramenta.
